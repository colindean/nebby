Nebby
=====

Nebby is a domain-specific language for describing surveys.

Nebby provides a quick, Ruby-throated way to construct a survey and present
the results in an easy-to-understand and easy-to-share way.

Requirements
------------

* Ruby 1.9+
* Sinatra
* MongoDB
